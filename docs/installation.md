# Installation — MDX Billing App

## Android APK

1. Download the latest APK from GitHub Releases.
2. Open the APK on your Android device.
3. Allow installation from trusted sources if required.
4. Install and open MDX Billing App.
5. Sign in and configure your shop profile.

## Notes

- Use only official APK files published in this repository.
- Do not install modified APK files from unknown sources.
